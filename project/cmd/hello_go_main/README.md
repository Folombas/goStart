## 📚 Что изучаем

### 1. Объявление переменных

**5 способов объявления:**

```go
// Способ 1: var с типом
var age int = 25

// Способ 2: var без типа
var name = "Alex"

// Способ 3: короткое объявление :=
city := "Moscow"

// Способ 4: множественное присваивание
var x, y = 10, 20

// Способ 5: множественное короткое
a, b := 5, 15
```

### 2. Константы

```go
const Pi = 3.14159
const MaxUsers = 100
```

### 3. Вывод на экран

```go
// Простой вывод
fmt.Println("Привет, Go!")

// Форматированный вывод
fmt.Printf("Имя: %s\n", name)
fmt.Printf("Уровень: %d\n", level)
```

### 4. Форматы вывода

| Спецификатор | Описание           | Пример           |
|--------------|-------------------|------------------|
| `%d`         | Целое число       | `42`             |
| `%s`         | Строка            | `"Hello"`        |
| `%v`         | Любое значение    | `true`           |
| `%.2f`       | Float с точностью | `3.14`           |
| `%t`         | Булево            | `true/false`     |
