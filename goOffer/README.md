# 🎯 Подготовка к собеседованию на Go Developer

Этот проект создан для систематической подготовки к собеседованиям на позицию Go-разработчика.

## 📁 Структура

```
goOffer/
├── README.md                    # Этот файл - план подготовки
├── theory/                      # Теоретические материалы
│   ├── basics.md               # Основы Go
│   ├── concurrency.md          # Конкурентность
│   ├── memory.md               # Управление памятью
│   └── advanced.md             # Продвинутые темы
├── practice/                    # Практические задачи
│   ├── easy/                   # Лёгкие задачи
│   ├── medium/                 # Средние задачи
│   └── hard/                   # Сложные задачи
├── interview_questions/         # Вопросы с собеседований
│   ├── junior.md               # Junior вопросы
│   ├── middle.md               # Middle вопросы
│   └── senior.md               # Senior вопросы
├── code_tasks/                  # Задачи на написание кода
│   ├── algorithms/             # Алгоритмы
│   ├── patterns/               # Паттерны
│   └── debugging/              # Отладка
└── cheatsheets/                # Шпаргалки
    ├── syntax.md               # Синтаксис
    ├── stdlib.md               # Стандартная библиотека
    └── best_practices.md       # Лучшие практики
```

## 🎓 Уровни подготовки

### Junior Go Developer
**Ожидания:**
- Знание синтаксиса и базовых конструкций
- Понимание основ конкурентности
- Умение писать простые тесты
- Базовая работа с Git

**Ключевые темы:**
- Переменные, типы, функции
- Слайсы, мапы, структуры
- Горутины и каналы (базово)
- Обработка ошибок

### Middle Go Developer
**Ожидания:**
- Глубокое понимание конкурентности
- Опыт работы с production кодом
- Понимание работы GC и оптимизация
- Знание популярных фреймворков

**Ключевые темы:**
- Паттерны конкурентности
- Профилирование (pprof)
- Мьютексы, атомарные операции
- Context и отмена операций
- Работа с БД и кэшем

### Senior Go Developer
**Ожидания:**
- Архитектурные решения
- Менторство и code review
- Оптимизация производительности
- Системный дизайн

**Ключевые темы:**
- Микросервисная архитектура
- gRPC, Protobuf
- Message queues (Kafka, RabbitMQ)
- Distributed systems patterns
- Performance tuning
